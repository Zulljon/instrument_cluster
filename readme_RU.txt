Имена файлов состоят из названия проекта и их назначения, например:
	Rover_Commander_v3.1a-B.Cu.gbr
		Rover_Commander_v3.1a - название проекта
		-B.Cu.gbr - нижний слой меди

соответственно:
	
	*-B.Cu.gbr  	-	задний слой меди
	*-B.Fab.gbr 	-	монтажные посадочные места сзади платы 
	*-B.Mask.gbr	-	паяльная маска сзади паты
	*-B.Paste.gbr	-	паяльная паста сзади паты
	*-B.SilkS.gbr	-	шелкография сзади паты
	*-B.CrtYd.gbr	-	области установки посадочных мест сзади платы
	*-B.Adhes.gbr	-	клей сзади платы

	*-F.Cu.gbr  	-	передний слой меди
	*-F.Fab.gbr 	-	монтажные посадочные места спереди платы
	*-F.Mask.gbr	-	паяльная маска спереди паты
	*-F.Paste.gbr	-	паяльная паста спереди паты
	*-F.SilkS.gbr	-	шелкография спереди паты
	*-F.CrtYd.gbr	-	области установки посадочных мест спереди платы
	*-F.Adhes.gbr	-	клей спереди платы

	*-Edge.Cuts.gbr	-	определение контура платы
	*.drl			-	файл сверловки
	*-drl_map.gbr	-	карта файла сверловки
